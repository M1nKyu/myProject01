# 기여 가이드

이 프로젝트는 포트폴리오 목적으로 공개되었습니다.

## 프로젝트 개요

eCarbon 은 웹사이트의 디지털 탄소 배출량을 분석하고 최적화 방안을 제시하는 플랫폼입니다.

## 주요 기능

- 탄소 배출량 분석
- 코드 최적화 분석
- 이미지 최적화
- 지속가능성 가이드라인 분석
- PDF 리포트 생성
- 다국어 지원 (4개 언어)

## 기술 스택

- Backend: Python, Flask, MongoDB, Redis, Celery
- Frontend: HTML, CSS, JavaScript, Chart.js
- DevOps: Docker, Nginx, Gunicorn
